##### JMonitor是一个监控系统，用于Java应用程序的监控。

##### JMonitor系统结构
这里有一段简单介绍

##### JMonitor支持的监控消息类型包括：
+  **Transaction**	主要用来记录某段代码的执行时间和次数。
+  **Event**	    主要用来记录某个事件发生的次数。
+  **Status**	    主要用来记录状态信息，包括CPU使用率、内存、堆、线程等信息。
+  **Problem**	    主要用来记录程序运行过程中的一些异常或者错误。
